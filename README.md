# TownyCore

**TownyCore** — это кастомный плагин-расширение для [Towny](https://github.com/TownyAdvanced/Towny), добавляющий уникальные механики поселений, экономики и зданий, основанные на ресурсах. Плагин создан для серверов, которые хотят предложить игрокам честный и увлекательный геймплей без донатного дисбаланса.

---

## Особенности

- **Создание города за ресурсы**  
  Для основания города требуются ресурсы (настраиваются в конфиге). Если ресурсов недостаточно — создание блокируется с понятным сообщением.

- **Приват территории за ресурсы**  
  Расширение территории города (команда `/townycore claim`) требует ресурсы, а не игровые монеты. Стоимость расширения растёт с каждым новым уровнем.

- **Уровни привата с формами территории**  
  Каждый уровень привата имеет свою форму (задаётся в конфиге в виде символьной матрицы). При повышении уровня территория города автоматически расширяется согласно заданной форме.

- **Экономика через здания**  
  Игроки могут строить специальные здания в городе:
  - Примеры:
      - **Шахта** — продажа руды.
      - **Ферма** — продажа сельхозпродукции.
  
  Постройка требует ресурсов. После постройки открывается меню скупки ресурсов за игровую валюту. Управление зданиями осуществляется через единое меню (`/townycore builds`).

- **Адаптивное GUI-меню зданий**  
  - Если здание не построено — показывает необходимые ресурсы и кнопку «Построить».
  - Если здание уже есть — кнопку «Открыть меню скупки».
  
  Внешний вид меню (иконки, названия, описания) полностью настраивается через конфиг.

- **Визуальные улучшения**  
  - Отображение границ города специальными блоками для наглядности.
  - При постройке здания отображаются препятствия, если блоки мешают или отсутствует опора.

---

## Команды

| Команда | Описание |
|---------|----------|
| `/townycore claim` | Расширить территорию города за ресурсы
| `/townycore builds` | Открыть меню зданий
| `/townycore build add` | (Админ команда) Добавить здание

---

## Установка и настройка

1. Скачайте последнюю версию плагина из [релизов]([https://github.com/your-repo/TownyCore/releases](https://github.com/veskmm/TownyCore/releases/tag/v)).
2. Поместите JAR-файл в папку `plugins` вашего сервера.
3. Убедитесь, что на сервере установлен Towny и все зависимости (Vault, Citizens — если используются).
4. Перезапустите сервер. Плагин создаст конфиг-файлы.
5. Настройте конфиги под свой сервер.
